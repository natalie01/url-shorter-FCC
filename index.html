FreeCodeCamp API Basejump: URL Shortener Microservice

User stories:

   I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.

  When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

  https://lackadaisical-antler.glitch.me/www.google.com

  https://lackadaisical-antler.glitch.me/http://www.google.com

  https://lackadaisical-antler.glitch.me/https://www.google.com

  https://lackadaisical-antler.glitch.me/google.com

Example creation output

  { "originalUrl":"www.google.com", "shortUrl":"https://lackadaisical-antler.glitch.me/8170" }

Usage:

  https://lackadaisical-antler.glitch.me/2871

Will redirect to:

  https://www.google.com/
